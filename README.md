# Covid19Tarama-BulanikMantik
Bulanik mantik kullanilarak Covid 19 taramasi yapilmasi

## Gerekli python modulleri:
 - [poetry](https://python-poetry.org/)
 - streamlit
 - [pyIT2FLS](https://github.com/Haghrah/PyIT2FLS)
 - numpy
 - pandas
 - matplotlib
